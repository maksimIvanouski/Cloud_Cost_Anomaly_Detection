# 🤖 Models Directory

This directory stores trained model artifacts generated by `src/train.py`.

## Model Files

| File | Description |
|---|---|
| `baseline_model.pkl` | Majority Class Baseline — always predicts the most frequent class (normal) |
| `logistic_regression_model.pkl` | Logistic Regression with `class_weight="balanced"` |
| `random_forest_model.pkl` | Random Forest optimized via GridSearchCV (cv=4, scoring=f1) |
| `isolation_forest_model.pkl` | Isolation Forest (unsupervised anomaly detector) |

## Important: Full Pipeline Artifacts

Each `.pkl` file contains the **full scikit-learn Pipeline** (preprocessor + classifier), not just the classifier alone. This means:

- The saved artifact includes all preprocessing steps (imputation, scaling, encoding)
- Loading the model and calling `.predict()` applies the exact same transformations used during training
- This ensures **consistent preprocessing during inference** and prevents train/serve skew

### Example Usage

```python
import joblib
import pandas as pd

# Load the full pipeline
pipeline = joblib.load("models/random_forest_model.pkl")

# Predict on new data — preprocessing is applied automatically
new_data = pd.DataFrame({...})
predictions = pipeline.predict(new_data)
```

## Reproducibility

All models are trained with `RANDOM_STATE=67` to ensure reproducible results across runs.

## Notes

- Model files (`.pkl`) are excluded from version control via `.gitignore`.
- To regenerate models, run `python src/train.py` after generating and processing the dataset.
- The Random Forest model typically achieves the best F1-score among the supervised models.
- The Isolation Forest model is unsupervised — it does not use labels during training.
